# 🧮 Calculator Using Streamlit

A simple and interactive calculator built using **Python** and **Streamlit**. This application allows users to perform basic arithmetic operations through an intuitive web interface.

## 🚀 Features

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly interface
- Handles division by zero gracefully
- Instant result display

## 🛠️ Tech Stack

- Python 3.x
- Streamlit

## 📂 Project Structure

```
Calculator Using Streamlit/
│── app.py
│── requirements.txt
└── README.md
```

## 📦 Installation

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Navigate to the project folder

```bash
cd "Calculator Using Streamlit"
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

or

```bash
python3 -m pip install -r requirements.txt
```

## ▶️ Run the Application

```bash
python3 -m streamlit run app.py
```

After running the command, Streamlit will automatically open the application in your default web browser.

If it doesn't open automatically, visit:

```
http://localhost:8501
```

## 📖 Usage

1. Enter the first number.
2. Enter the second number.
3. Select an arithmetic operation.
4. Click the **Calculate** button.
5. View the calculated result instantly.
